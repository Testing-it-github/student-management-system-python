# student-management-system-python
A CLI-based Student Management System built with Python that stores and manages student records using CSV file handling. Supports Add, View, Search, Update, and Delete operations.
## 📋 Features
- ➕ Add Student
- 👁️ View All Students
- 🔍 Search Student by ID or Name
- ✏️ Update Student Details
- 🗑️ Delete Student Record
- ✅ Input Validation (ID, Name, Course, Mobile)

## 🛠️ Technologies Used
- Python
- CSV File Handling
- Functions, Loops, Conditions

## 📁 Data Storage
All data is stored in `students.csv` with columns:
ID, Name, Course, Mobile Number

## ▶️ How to Run
1. Clone the repository
2. Open terminal in project folder
3. Run command: python students.py
